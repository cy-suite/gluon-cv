## A Mini Implementation Of Cyclegan


**Prerequisites**
1. Python 3.6+
2. Gluoncv
3. Mxnet


**Download  Dataset**
```bash
python extract_data.py
```

**Run The Model**
```bash
python Cyclegan.py --num_epoch 5 --lr 0.0002
```
